# 亚马逊广告分析工具

自动分析PPC广告数据，优化广告投放。

## 功能

- ACOS分析：自动计算广告投入产出比，诊断异常
- 关键词优化：识别高转化词、高花费低转化词
- 出价策略建议：根据数据调整竞价
- 广告位分析：搜索结果顶部/商品页面/其余位置
- 否定关键词建议：排除无效点击

## 使用示例

告诉Agent：
```
分析这份广告报告：[粘贴广告数据]
我的目标ACOS是20%
```

返回：
- 当前ACOS vs 目标
- 表现最好的3个关键词
- 建议否定关键词
- 出价调整建议
